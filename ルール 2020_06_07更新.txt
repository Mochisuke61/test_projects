■命名規則

　・ファイル   - アッパーキャメルケース ( 例 : TestNameCase.cpp ) 
　・フォルダ名 - アッパーキャメルケース ( 例 : TestNameCase) )
　・関数名     - アッパーキャメルケース ( 例 : TestNameCase() )
　・変数名     - ロワーキャメルケース   ( 例 : int testNameCase )
  ・クラス名   - アッパーキャメルケース ( 例 : class TestNameCase{}; )
　・メンバ変数 - ロワー + 先頭にアンダーバー ( 例 : int _testNameCase )
